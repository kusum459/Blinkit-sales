Blinkit Sales Analysis Dashboard

Project Overview

This project presents an interactive business intelligence dashboard built in Microsoft Excel to analyze sales performance for Blinkit, India's quick-commerce grocery delivery platform. The dashboard consolidates outlet-level sales, product category, and store format data into a single visual interface, allowing users to explore performance across regions, outlet types, and product categories.

Objective

The goal of this project is to convert raw grocery sales data into actionable business insights that support decision-making around product performance, outlet efficiency, and regional sales trends.

Key Metrics

Total Sales: 1.2M
Average Sales: 141k
Number of Items: 8523
Average Rating: 3.9

Dashboard Features

Total Sales by Fat Content
Pie chart showing the split between Low Fat products and Regular products.

Fat Content by Outlet
Column chart comparing Low Fat versus Regular product sales across Tier 1, Tier 2, and Tier 3 outlets.

Total Sales by Item Type
Horizontal bar chart ranking sales across categories such as Fruits and Vegetables, Snack Foods, Household, Frozen Foods, Dairy, Canned goods, and others.

Sales by Outlet Size
Donut chart showing distribution across High, Medium, and Small outlet sizes.

Sales by Outlet Location
Bar chart comparing sales across Tier 1, Tier 2, and Tier 3 locations.

Count of Outlet Type
Column chart showing the number of outlets by format, including Grocery Store, Supermarket Type 1, Type 2, and Type 3.

Sum of Outlet Type
Bar chart showing total sales aggregated by store format.

Filters

The dashboard includes interactive filters built using Excel slicers, allowing users to drill down into specific segments of the data.

Outlet Size: High, Medium, Small
Item Type: Hard Drinks, Health and Hygiene, Household, Meat, Others, Seafood, Snack Foods, Soft Drinks, Starchy Foods
Outlet Location Type: Tier 1, Tier 2, Tier 3

Tools Used

Microsoft Excel
Pivot tables and pivot charts
Slicers for interactive filtering
Excel formulas for data calculations
Conditional formatting and dashboard design elements

Insights

Regular fat content products contribute a larger share of total sales compared to low fat products.
Tier 3 outlets generate the highest sales among the three outlet location tiers.
Supermarket Type 1 accounts for the highest count among outlet types.
Sales performance varies noticeably by outlet size, with medium sized outlets showing a significant share of total sales.

Conclusion

This dashboard demonstrates the use of Excel based data visualization techniques to transform raw retail data into a clear, interactive reporting tool. It highlights key business metrics and trends that can support strategic planning for outlet management, product stocking, and regional sales strategy.
